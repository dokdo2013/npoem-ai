# N행시 AI Generator

OpenAI API를 활용한 N행시 AI Generator

## Local Development Guide

```sh
# Install Pnpm
npm i -g pnpm

# Install dependencies
pnpm i

# Start Development Server (8080: web, 8081: api)
pnpm dev

# Start Web Development Server
pnpm web dev

# Start Api Development Server
pnpm api dev
```
